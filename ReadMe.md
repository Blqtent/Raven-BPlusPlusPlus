How To Setup The Workspace Yourself?                                                                                                                                                                                                   
Download The SRC,
Put The SRC In Some Random Folder,
Open CMD In The Folder's Path & Type " gradlew SetupDecompWorkspace && gradlew idea "
Open The Project In Intellij (Not Eclipse)
